RESUMO COMPLETO DO PROJETO WK CRM MICROSERVICES
 VISÃO GERAL DO PROJETO
WK CRM é uma plataforma empresarial completa de microserviços para gestão de relacionamento com clientes, usando arquitetura moderna e tecnologias de ponta.

 ARQUITETURA DO SISTEMA
 Frontend Applications
Admin Dashboard: Angular 18 (wk-admin-frontend) - Interface administrativa
Customer Portal: Vue 3 (wk-customer-app) - Portal do cliente
Simple Admin: HTML/CSS/JS (wk-admin-simple) - Interface básica
 Backend APIs
Main CRM API: Laravel 11 (wk-crm-laravel) - API principal com DDD
Secondary CRM: .NET 8 (wk-crm-dotnet) - Demo Clean Architecture
Products API: Node.js (wk-products-api) - Gestão de produtos
API Gateway: Node.js (wk-gateway) - Centralizador de APIs
AI Service: Python FastAPI (wk-ai-service) - IA com Google Gemini
 Infrastructure
Database: PostgreSQL + Redis
Web Server: NGINX (proxy reverso)
Containerization: Docker Compose
CI/CD: GitHub Actions
